## 회귀분석 , 회귀식이란? 

**회귀(regression)** 라는 말의 뜻은 무엇일까?  사전적 의미는 'go back to an earlier and worse condition' 이라는 뜻으로 <br>
이전의 대표적인 상태로 돌아간다는 의미이다.
